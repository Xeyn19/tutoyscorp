# Tutoy Corp Website User Flow

```text
START
  ↓
User opens Tutoy Corp Website
  ↓
Landing Page Loads
  ↓
Display:
- Header (Logo, Navigation)
- Hero Section (Tagline)
- Features Section
- Mission Section
- Audience Section
- Pricing Preview
- Footer
  ↓
User Action?
  ├── Click "Connect With Us"
  │       ↓
  │   Go to Contact / Inquiry Page
  │       ↓
  │   Display Form Fields:
  │      - Full Name
  │      - Email Address
  │      - Contact Number
  │      - Company Name (optional)
  │      - Selected Service/Product
  │      - Message / Requirements
  │       ↓
  │   User fills up form
  │       ↓
  │   Click "Submit"
  │       ↓
  │   Show Success Message
  │       ↓
  │   Store data in database
  │       ↓
  ├── Click "Back to Landing Page"
  │       ↓
  │   Go back to Landing Page
  │
  ├── Click "View Plans"
  │       ↓
  │   Go to Pricing Section
  │
  └── Exit Page
          ↓
         END
```
